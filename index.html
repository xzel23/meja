Meja Spreadsheet library

Meja is a Java library for handling spreadsheet and table data.

Overview

It can be used for:
  • loading and manipulating Excel and CSV files.
  • displaying spreadsheets in Java applications

Documentation

The API-documentation is generated from the Javadocs.

Installation

  1. Get the source by cloning https://github.com/xzel23/meja.git.
  2. Change into the meja directory and invoke ant to build the library. Dependencies will be downloaded automatically using Apache Ivy which also will be downloaded automatically when building for the first time.
  3. Change into the meja-samples directory and invoke ant to build the sample application.

Meja comes with project data for both Eclipse and Netbeans.
